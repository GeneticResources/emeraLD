# emeraLD
tools to efficiently retrieve and calculate LD 

# to install
with libStatGen already present:
     git clone https://github.com/statgen/emeraLD.git
     cd emeraLD
     make LIBSTATGEN_PATH=my/path/to/libStatGen
without libStatGen:
     git clone https://github.com/statgen/emeraLD.git
     cd emeraLD
     make cloneLib
     make
